## 课题一：全国天气气候分析系统

### 1. 系统简介

全国天气气候分析系统是一个整合多个城市气象数据的综合分析平台，通过对温度、湿度、降水量、风速、气压等气象要素的长期趋势分析，揭示气候变化规律。系统能够进行季节对比分析、极端天气识别、地域气候特征研究，为气象部门、农业规划和公众出行提供可靠的气象数据服务。

### 2. 技术要求和限制条件

（1）通过 Python 调用天气 API 完成多城市历史气象数据采集与数据清洗

（2）利用 HDFS 进行气象数据的分布式存储

（3）基于 Spark RDD 编程模型完成各城市平均温度、最高/最低温记录、降水分布等统计分析

（4）将分析结果存储至 MySQL 数据库，结合 ECharts 实现温度趋势折线图、降水量柱状图等可视化展示
